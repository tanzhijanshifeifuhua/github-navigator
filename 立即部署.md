## 🎯 部署步骤（只需 3 步）

### 第 1 步：打开 Railway
在浏览器中访问：**https://railway.app**

### 第 2 步：登录并创建项目
1. 点击右上角 **"Login"**
2. 选择 **"Login with GitHub"**
3. 授权 Railway 访问你的 GitHub
4. 点击 **"New Project"**
5. 选择 **"Deploy from GitHub repo"**
6. 找到并点击 **"github-navigator"**

### 第 3 步：等待部署完成
- Railway 会自动检测 Python 项目
- 自动安装依赖（requirements.txt）
- 自动启动应用（使用 Procfile）
- 大约 2-3 分钟完成

---

## 🌐 获取访问地址

部署完成后：
1. 在 Railway 项目页面，点击你的服务
2. 点击 **"Settings"** 标签
3. 找到 **"Domains"** 部分
4. 点击 **"Generate Domain"**
5. 复制生成的地址（例如：xxx.up.railway.app）

## 💡 提示

### 如果部署失败
1. 查看 Railway 的部署日志
2. 检查是否有错误信息
3. 确认 github_projects.json 文件已提交

### 如何更新数据
1. 本地运行 `scripts\crawler.py`
2. 运行 `scripts\add_chinese_descriptions.py`
3. 提交并推送更新
4. Railway 会自动重新部署

### 查看日志
在 Railway 项目页面：
- 点击 **"Deployments"**
- 选择最新部署
- 查看实时日志
